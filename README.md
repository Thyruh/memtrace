# Memory Trace - is a nano library with a valgrind-like functionality and dead easy API with three functions.

In case your lsp screams at function definitions in a header file, let me know, or pull request a header guard.
I added an experimental for me .clandg which turned those errors off for me. Don't know if that is just a "Works on my machine" situation.

# To compile use nob: 

```
cc -onob nob.c && ./nob
```

# And run: 
```
./build/main
```
